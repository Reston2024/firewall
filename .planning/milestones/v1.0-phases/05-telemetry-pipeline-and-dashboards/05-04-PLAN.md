  Panel 1 — DASH-02: Firewall Drops Over Time
    Type: Time series (bar chart)
    LogQL: sum(count_over_time({job="ipfire-syslog"} |= "FORWARDFW" [1m]))
    Title: "Firewall Drops per Minute"

  Panel 2 — DASH-04a: Top Blocked Source IPs (24h)
    Type: Table
    LogQL:
      topk(10,
        sum by (src_ip) (
          count_over_time(
            {job="ipfire-syslog"} |= "FORWARDFW"
            | regexp `SRC=(?P<src_ip>\d+\.\d+\.\d+\.\d+)`
            [24h]
          )
        )
      )
    Note: src_ip is extracted at query time (not a Loki label) on the syslog stream — use regexp stage

  Panel 3 — DASH-04b: Top Triggered Suricata Rules (24h)
    Type: Table
    LogQL:
      topk(10,
        sum by (signature) (
          count_over_time({job="suricata-eve", event_type="alert"}[24h])
        )
      )
    Note: signature IS a Loki label on the suricata-eve stream (set in Alloy eve_parse pipeline)

  Panel 4 — DASH-01: Threat Trace — IP Activity (syslog + IDS)
    Type: Logs
    LogQL: {job=~"ipfire-syslog|suricata-eve"} |= "$src_ip"
    Variable: src_ip (text box variable, user inputs IP to trace)
    Title: "Threat Trace: $src_ip across Firewall + IDS"
    Note: This is a single LogQL query that unions both streams by the variable IP
