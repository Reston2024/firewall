**Step 1: Pre-flight checks**
```bash
docker --version && docker compose version
python3 --version
ss -tlnp | grep -E ':443|:5601|:9200|:5044'
ss -ulnp | grep ':514'
```
Verify Docker >= 26.x, Docker Compose v2, Python >= 3.9. Note which ports are in use. Port :514 UDP will likely be held by rsyslog — this is expected and OK for Phase 9 (syslog ingestion deferred to Phase 10 per research constraint). Do NOT attempt to stop rsyslog or reconfigure :514 in this phase.

If :443 or :5601 or :9200 are in use by something unexpected, STOP and report the conflict.

**Step 2: Kernel parameter**
```bash
sudo sysctl -w vm.max_map_count=262144
echo 'vm.max_map_count=262144' | sudo tee /etc/sysctl.d/99-malcolm.conf
```

**Step 3: Swap (4GB)**
Check if swap already exists: `free -m | grep Swap`. If Swap total >= 3000, skip this step.
Otherwise:
```bash
sudo fallocate -l 4G /swapfile
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile
echo '/swapfile none swap sw,pri=10 0 0' | sudo tee -a /etc/fstab
```

**Step 4: Clone Malcolm**
```bash
sudo git clone --depth 1 --branch v26.02.0 https://github.com/cisagov/Malcolm.git /opt/malcolm
sudo chown -R opsadmin:opsadmin /opt/malcolm
```

**Step 5: Run auth_setup and install.py**
```bash
cd /opt/malcolm
python3 scripts/auth_setup
```
When prompted, create admin credentials. Record the password for use in validation script and OpenSearch API calls.

```bash
python3 scripts/install.py
```
Accept defaults: run with "malcolm" profile. Accept all other defaults.

**Step 6: CRITICAL — Edit heap and config files BEFORE first start**

After `install.py` completes, inspect the generated config directory:
```bash
ls /opt/malcolm/config/
```
Identify the env files. Then edit:

In the opensearch env file (likely `/opt/malcolm/config/opensearch.env` or similar — find the file containing `OPENSEARCH_JAVA_OPTS`):
```
OPENSEARCH_JAVA_OPTS=-Xms6g -Xmx6g
OPENSEARCH_BOOTSTRAP_MEMORY_LOCK=false
```

In the logstash env file (likely `/opt/malcolm/config/logstash.env` or similar — find the file containing `LS_JAVA_OPTS`):
```
LS_JAVA_OPTS=-Xms1g -Xmx1g
```

**Step 7: Disable live capture**

In the pcap-capture env file:
```
PCAP_ENABLE_NETSNIFF=false
PCAP_ENABLE_TCPDUMP=false
```

In the zeek env file:
```
ZEEK_LIVE_CAPTURE=false
ZEEK_ROTATED_PCAP=false
```

In the suricata env file:
```
SURICATA_LIVE_CAPTURE=false
```

**Step 8: Set storage pruning env var**

In the opensearch env file, also add:
```
OPENSEARCH_INDEX_SIZE_PRUNE_LIMIT=200g
```

In the arkime env file:
```
MANAGE_PCAP_FILES=true
ARKIME_FREESPACEG=15%
```

**Step 9: Pull images**
```bash
cd /opt/malcolm
docker compose --profile malcolm pull
```
This downloads ~10-15GB of container images. May take 10-30 minutes depending on bandwidth.

**Step 10: Start Malcolm**
```bash
cd /opt/malcolm
./scripts/start
```

**Step 11: Monitor startup (2-5 minutes)**
```bash
./scripts/logs  # watch logs, Ctrl+C after 2-3 min
docker compose ps  # verify containers are coming up
free -m  # verify RAM not exhausted
```

Wait until `docker compose ps` shows containers as healthy (except possibly Arkime capture).

**Step 12: Post-start verification**
```bash
free -m  # used should be < 14000 MB
sudo dmesg | grep -i oom | tail -20  # should be empty
docker stats --no-stream  # verify OpenSearch container RSS ~ 6-8GB
curl -sk -o /dev/null -w '%{http_code}' https://localhost:5601/  # should be 200 or 302
```

**IMPORTANT NOTES:**
- Do NOT attempt to configure Malcolm's syslog input on :514 — rsyslog holds that port and will conflict. Syslog ingestion is Phase 10 scope.
- Do NOT stop the existing Loki/Grafana stack at /opt/telemetry — it stays running per v2.0 architecture decision.
- The `install.py` and `auth_setup` scripts are interactive — they require terminal input. Run them in an interactive SSH session, not via piped commands.
