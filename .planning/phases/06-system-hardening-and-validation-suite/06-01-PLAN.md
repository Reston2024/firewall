```
net.ipv4.conf.default.accept_redirects = 0
net.ipv4.conf.all.rp_filter = 1
net.ipv4.conf.default.rp_filter = 1

# Additional CIS params safe for router:
net.ipv4.tcp_syncookies = 1
net.ipv6.conf.all.accept_redirects = 0
net.ipv6.conf.default.accept_redirects = 0
```

**3. Create `scripts/validate-reboot.sh`** (per D-15, D-16, D-17):

Two-mode script:
- `--snapshot` mode: Capture pre-reboot state to `/root/reboot-snapshot.txt`:
  - sysctl hardened values (all params from sysctl-hardening.conf)
  - Listening services: `ss -tlnp` output
  - Full iptables ruleset hash: `iptables-save | sha256sum`
  - Config file hashes: `sha256sum` of all monitored files
  - Print instructions: "Now reboot: reboot" and "After reboot run: bash validate-reboot.sh --compare"

- `--compare` mode: Capture current state in same format, diff against stored snapshot
  - If no diff: PASS + "System state identical after reboot"
  - If diff found: print diff, FAIL + "State changed after reboot — review diff above"
  - Exit 0 if identical, exit 1 if different

Script must use the same pass()/fail() pattern. Include date stamps in snapshot for traceability. Do NOT trigger reboot — manual only per D-17.
